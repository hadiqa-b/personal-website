# hadiqabasit.com

Personal portfolio website for Hadiqa Basit — ML Engineer.

Built with Astro, Tailwind CSS v4, and React. Deployed on Cloudflare Workers.

## Stack

- **Framework**: Astro 6
- **Styling**: Tailwind CSS v4 (Gruvbox dark theme)
- **Background**: ShaderGradient (Three.js / React Three Fiber)
- **Hosting**: Cloudflare Workers

## Development

```sh
npm install
npm run dev
```

Dev server runs at `localhost:4321`.

## Deployment

```sh
npm run deploy
```

Builds the site and deploys to Cloudflare via Wrangler.
